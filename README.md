# Tổng quan Đồ án Cinema-SA

Cinema-SA là một phần mềm quản lý hệ thống đặt vé xem phim. Hệ thống cho phép khách hàng xem những bộ phim đang chiếu và sắp chiếu. Khi đặt vé thì yêu cầu phải đăng nhập. Ngoài ra, hệ thống còn có admin để quản lý phim, vé và khách hàng.

## Công nghệ sử dụng

- Frontend: React
- Backend: Flask
- Database: MySQL
